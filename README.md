## Usage

### Requirements

* Visual Studio 2017
* oF v0.11.0
* Python 3.7
* pybind11 2.4.3

### Build `pyof`

1. Set a path to your python to env `PYTHONPATH`
2. Put this project into `<oF_root>/apps/myApps`
3. Open `pyof.sln` and run build
